Instrucciones

1: Para iniciar y pausar el juego presionar la barra espaciadora.
2: Para mover el primer jugador se presiona las teclas arriba y abajo.
3: Para mover el segundo jugador se presiona las teclas "w" y "s".
4: hay un control de dificultad, en el cual se puede escoger cualquiera de las 3 opciones.
5: El juego termina cunado uno de los dos deje paras la pelota:
